# No title MOD

このMODは、Yuyutoが「面白そう」「住みたい」「ずっと居たい」を詰め込んだ総合要素追加MODです。
このMODのコンセプトは「Yuyutoが世界に欲しいと思うもの」。

具体的な内容は、\
工業があり、魔術があり、それらが自然に混ざったもの
戦闘がアクション+FPSで探索する世界も上下差があり、新しいディメンションもある
フィールドにオペレーションフィールドが存在し、建材もあり、かわいい動物とかモンスターとかもあり、たまにホラーもあるMODです。

「そんな行き当たりばったりのなんでもあるけれど緩やかで穏やかで時に激しく時に感動し時に幻想的な世界になり時にホラーが起きる」\
…こんな世界。\
このMODは日本語にのみ対応しています。

# 1.工業
工業はマルチブロックとかも入れつつ初心者でもとっつきやすいしガチ勢もそれなりに時間がかかるバランスで、
MekanismやIndustrial Craft 2、Createのような中型・大型工業MODを参考に
FEとは全く違うリアリティなエネルギーシステムで動くという感じで制作しました。

コンセプトは　**「大量生産」「ロマン」「ちょっとリアル」**。どんな工業をしたい？
